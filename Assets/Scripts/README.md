# 宇宙射擊遊戲腳本系統

這是一個完整的3D宇宙射擊遊戲腳本系統，包含玩家控制、武器系統、敵人AI、生命值管理和分數系統。

## 腳本列表

### 核心腳本
- **PlayerController.cs** - 玩家控制器，處理6自由度移動和輸入
- **WeaponSystem.cs** - 武器系統，支援多種射擊模式和物件池
- **Projectile.cs** - 飛彈行為，包含追蹤和視覺效果
- **PlayerHealth.cs** - 玩家生命值系統，包含無敵時間和復活功能
- **EnemyHealth.cs** - 敵人生命值系統，包含分數獎勵
- **EnemyController.cs** - 敵人AI控制器，多種移動模式
- **ScoreManager.cs** - 分數管理系統，包含最高分記錄

## 快速設置

### 1. 玩家設置
1. 創建玩家物件，添加以下組件：
   - `PlayerController`
   - `WeaponSystem`
   - `PlayerHealth`
   - `Rigidbody`
   - `Collider`

2. 設置標籤為 "Player"

### 2. 敵人設置
1. 創建敵人物件，添加以下組件：
   - `EnemyController`
   - `WeaponSystem` (可選)
   - `EnemyHealth`
   - `Rigidbody`
   - `Collider`

2. 設置標籤為 "Enemy"

### 3. 飛彈設置
1. 創建飛彈預製體，添加以下組件：
   - `Projectile`
   - `Rigidbody`
   - `Collider` (設置為Trigger)

### 4. 分數系統
1. 在場景中創建一個空物件
2. 添加 `ScoreManager` 組件

## 主要功能

### 玩家控制
- **移動**: WASD (前後左右) + Q/E (上下)
- **射擊**: 空白鍵
- **滑鼠瞄準**: 自動瞄準滑鼠位置

### 武器系統
- **單發射擊**: 基本射擊模式
- **多發射擊**: 同時發射3發子彈
- **散彈射擊**: 扇形散彈模式
- **物件池**: 優化性能

### 敵人AI
- **直線移動**: 基本移動模式
- **圓形移動**: 繞圈移動
- **正弦波移動**: 波浪式移動
- **螺旋移動**: 螺旋軌跡
- **追蹤玩家**: 智能追蹤模式

### 生命值系統
- **傷害處理**: 自動傷害計算
- **無敵時間**: 受傷後短暫無敵
- **視覺效果**: 受傷閃爍效果
- **復活功能**: 玩家可復活

### 分數系統
- **即時分數**: 當前遊戲分數
- **最高分記錄**: 自動保存最高分

## 注意事項

1. **Rigidbody設置**: 確保所有物件都有適當的Rigidbody設置
2. **標籤設置**: 正確設置Player和Enemy標籤
3. **碰撞器**: 飛彈使用Trigger碰撞器避免物理碰撞

這個腳本系統提供了完整的3D射擊遊戲基礎，可以根據需要進行擴展和自定義。
